# 📊 IPL RCB Analysis

# Objective
To analyze Royal Challengers Bangalore (RCB) performance using SQL and Excel and derive meaningful insights and recommend new players for the team.

# Tools Used
- SQL (Joins, Aggregations, CTEs)
- Excel (Data cleaning, analysis)

# Key Insights

- RCB's top performers contributed majority of total runs
- Middle-order inconsistency affected match outcomes
- Certain seasons showed significant performance dips
- Win rate improved when top-order performed well

# Project Highlights
- Used complex SQL queries for data extraction
- Performed data analysis using Excel
- Generated actionable insights from raw data
